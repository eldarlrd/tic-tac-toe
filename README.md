> Tic Tac Toe is a classic game built using
> - **[Inferno](https://infernojs.org)**
> - **[Tachyons](https://tachyons.io)**
> - **[Flow](https://flow.org)**
> - **[webpack](https://webpack.js.org)**
>
> with PvP and PvE modes.

## Install
```sh
yarn
```
## Develop
### Run Env.
```sh
yarn start
```
### Prod. Build
```sh
yarn build
```
### Type Check
```sh
yarn type
```
### Format Code (ESLint + Prettier)
```sh
yarn format
```
## Deploy
```sh
yarn deploy
```
